dir_absolute_path

Prints the absolute path of the input directory. Depends on: GNU readlink, "greadlink" on OS X and FreeBSD 

function dir_absolute_path
Parameters
$dir_name

The directory name for which you need an absolute path without the ending newline

Return Value

The exit code of the readlink execution

Discussion

Example:

# print the absolute path of the current directory
dir_absolute_path .

dir_cleanup

Recursive delete of all the directories and their contents having the input name into the target directory 

function dir_cleanup
Parameters
$target_dir

The target directory containing all the subdirectories having the same name that you wish to remove

$dir_name

The directory name that you wish to remove

Return Value

The exit code find | xargs execution

Discussion

Example:

# cleanup a SVN working copy
dir_cleanup "${working_copy_path}" ".svn"

dir_exists

Checks if a directory exists 

function dir_exists
Parameters
$dir_path

The directory path to check

$no_print

Don't print the output. Defaults to falsy

Return Value

The exit code of test -d

Discussion

Example:

if [ $(dir_exists "${dir_path}") -eq $FALSE ]
then
	# do something about the missing directory
fi

Notice: if the performance is a concern, use if with the -d flag. This function favors readability.
