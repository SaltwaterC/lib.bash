file_cleanup

Recursive delete of all the files having the input name into the target directory 

function file_cleanup
Parameters
$target_directory

The target directory containing the files with the name that you wish to remove

$file_name

The file name that you wish to remove

Return Value

The exit code of find | xargs

Discussion

Example:

# remove all the .gitignore files from the current directory and subdirectories
file_cleanup . .gitignore

file_exists

Checks if a file exists 

function file_exists
Parameters
$file_path

The file path to check

$no_print

Don't print the output. Defaults to falsy

Return Value

The exit status of test -e

Discussion

Example:

if [ $(file_exists "${file_path}") -eq $FALSE ]
then
	# do something about the missing file
fi

Notice: if the performance is a concern, use if with the -e flag (or any appropriate file related flag). This function favors readability.

Discussion

Example:
